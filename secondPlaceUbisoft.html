2nd Place Ubisoft NEXT 2023 Programming

In 2023 I participated in the Ubisoft NEXT 2023 programming competition. For this competition Ubisoft I had one weekend to make a game, I was also provided a simple 2D graphics API and a month to prepare. During that month I implemented an Entity Component System architecture for my game engine. I also implemented a math library to use the 2D API to create a 3D engine. I was then able to create a two player 3D Bomber-man game with a dynamic camera. Below is a video of the game where I describe the systems I implemented to create it.
