Tower Defense Game

This was my submission for the Ubisoft Next coding competition. The challenge was to build a tower defense game using C++ in two days. I used boids flocking algorithm to create the movement of the enemys. I used simple 2d inverse kinematics to create the snake movement. I created some simple computational geometry functions to help witht the creation of game objects. Unfortunatly my submission was not selected, but I am still very happy with the end result.
