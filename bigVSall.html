The Big VS Them All

The goal of this project was to create an online asymmetrical PVP game where one player controls a MOBA style kaiju and the other controls an RTS style army. I completed this project with a 6 person team. My main role was gameplay programming for this project. I opted to implement the Mixin design pattern to create an easy to use, data driven system for modularity. I then created a custom interface to allow designers to create and modify gameplay features without needing to change the code. We will be displaying the Game at the CNE August 30 and 31.
